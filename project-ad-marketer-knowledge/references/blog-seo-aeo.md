# Blog SEO/AEO Writing Reference

Use this reference when writing Korean marketing blog content for GabiaCNS.

## Format Rules

Use the blog editor format intentionally:

- Title = H1 equivalent
- Main sections = H2 / "소제목" or "제목 2"
- Subsections = H3 / "제목 3"
- Add a key summary near the top.
- Use tables for comparisons.
- Use FAQ for AEO.
- Add official or credible sources when factual claims require support.
- Add image/screenshot insertion points.
- End with a practical CTA.
- Suggest tags.

## Image Rules

Prefer direct image upload through the blog editor.

Avoid:

- Copy-pasting images from other posts
- Hotlinking external images from Imgur/Unsplash/etc.
- Using images without rights or attribution
- Overusing AI-looking generic images

For blog visuals:

- Use screenshots when explaining platform UI.
- Use tables or diagrams when explaining concepts.
- Use simple branded images for summaries.
- Keep image file names descriptive when possible.

## Tone

GabiaCNS blog tone:

- Practical and easy to understand
- Professional but not stiff
- Explain official changes in Korean business context
- Include "주의할 점", "운영 포인트", "실전 시나리오", "FAQ", "마치며" where useful
- Avoid direct translation tone

## AEO Checklist

- Put the answer early.
- Use question-style headings when searchers likely ask questions.
- Include FAQ.
- Define terms plainly.
- Add tables where comparison helps.
- Use examples and scenarios.
- Avoid vague marketing claims without explanation.
