# Gabia Codex Ad Skills

Codex에서 한국 온라인 광고 제안, 매체 전략, 네이버/카카오/Google/Meta 운영 논리, WSEKR류 교육 리드 제안, SEO/AEO 블로그 초안 작성에 활용하기 위한 Skill 패키지입니다.

## Included Skills

- `online-ad-marketing-advisor`
- `project-ad-marketer-knowledge`

## Install

새 PC에서 이 저장소를 내려받은 뒤, 두 Skill 폴더를 Codex Skill 경로에 복사합니다.

Windows 기본 경로:

```powershell
Copy-Item -Path .\online-ad-marketing-advisor -Destination "$env:USERPROFILE\.codex\skills" -Recurse -Force
Copy-Item -Path .\project-ad-marketer-knowledge -Destination "$env:USERPROFILE\.codex\skills" -Recurse -Force
```

설치 후 Codex를 새로 열면 Skill 목록에 반영됩니다.
